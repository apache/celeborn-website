    Apache Celeborn™ 0.5.0 Release Notes

    Highlight

    • Support TLS for internal communication
    • Support Flink 1.19
    • Support Java 21
    • Support memory storage(experimental)
    • Memory usage and performance optimization
    • Enhanced metrics and HTTP APIs
